# Local development server

This small Express server serves the `codes/` folder and provides a demo `/login` endpoint.

Quick start (Windows):

```powershell
cd "c:\Users\HP\Desktop\krish chaudhary"
npm install
npm start
```

Open http://localhost:3000/login.html
